# FIRST_SHELL_SCRIPT
This shell script lets the user to setup the LAMP STACK based on the distro they are using. Currrently added debian and red hat

#### ./scriptname.sh [args 1-4] 
#### args - functionalities

First Header | Second Header
------------ | -------------
1 | INSTALL
2 | START SERVICE
3 | STOP SERVICES
4 | BACKUP SQL
DEFAULT | GIT CLONE
